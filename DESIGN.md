# ExamAI — DESIGN.md

מפרט עיצוב לריברנד מלא. סגנון: **מינימליזם חמים, כותרות Serif, משטחים רכים**
(בהשראת Notion — getdesign.md/notion). המטרה: מראה אֶדִיטוֹריאלי ואנושי, **לא** מראה "AI סגול גנרי".

> שימוש: זהו מקור האמת לעיצוב. כל ערכי הצבע/פונט/מרווח בקוד צריכים להתיישר לכאן.
> בקוד הצבעים מרוכזים באובייקט `C` בכל קומפוננטה — מעדכנים שם.

---

## 1. פילוסופיית עיצוב

- **חמים, לא קר.** רקעים בגוון קרם/שמנת ולא לבן כחלחל. אפורים חמים.
- **אֶדִיטוֹריאלי.** כותרות בפונט Serif (רגש של ספר/מאמר), גוף בסאנס נקי.
- **שקט.** מעט צבע, הרבה מרחב נשימה, גבולות עדינים מאוד, צללים כמעט בלתי-מורגשים.
- **אנושי.** אייקוני קו (lucide) במקום אימוג'ים. פינות מעוגלות מתונות, לא pill.
- **ביטחון רגוע.** הצבע הראשי כהה ומאופק (דיו), לא ניאון. אקסנט אחד חם בלבד.

## 3. טיפוגרפיה

### פונטים (עברית, Google Fonts)
- **כותרות (Serif):** `Frank Ruhl Libre` — serif עברי קלאסי ויפהפה. fallback: Georgia, serif.
- **גוף (Sans):** `Assistant` או `Heebo` — נקי, קריא, מודרני. fallback: system-ui.
- **מספרים/קוד:** `ui-monospace`, monospace (לקודי גישה).

הטענה ב-`index.html`:
```html
<link href="https://fonts.googleapis.com/css2?family=Frank+Ruhl+Libre:wght@500;700;900&family=Assistant:wght@400;500;600;700&display=swap" rel="stylesheet">
```

## 7. כפתורים
- **ראשי:** רקע `ink` (#37352F), טקסט קרם (#FAF9F6), radius 8px, padding `11px 20px`, משקל 600. ללא gradient.
- **אקסנט (CTA חם):** רקע `accent` (#C2683D), טקסט לבן. לשימוש מדוד — CTA מרכזי בלבד.
- **משני:** רקע שקוף, גבול `border`, טקסט `text`.
- **רפאים:** טקסט בלבד בצבע `muted`, hover מוסיף `surfaceAlt`.
- hover: הכהיה עדינה + `translateY(-1px)`. transition 0.15s.

## 8. אייקונים
- **`lucide-react` בלבד** (כבר מותקן). קו 1.75px, גודל 16–20px, צבע `muted`/`text`.
- **להסיר את כל האימוג'ים** מהממשק (🤖📋✏️🚀📂♿...) ולהחליף באייקוני lucide:
  - 🤖 עוזר → `Sparkles` / `MessageCircle`
  - 📋 מבחנים → `ClipboardList`
  - ✏️ עריכה → `Pencil`
  - 🗑 מחיקה → `Trash2`
  - 📂 מבחני עבר → `FolderOpen`
  - 👥 תלמידים → `Users`
  - ⚙️ הגדרות → `Settings`
  - ♿ נגישות → אפשר להשאיר אייקון נגישות סטנדרטי (זיהוי)

## 9. עקרונות לייאאוט
- יישור לימין (RTL), כותרות Serif בולטות, הרבה whitespace.
- כרטיסים על רקע קרם עם גבול עדין במקום צל בולט.
- ה-landing: hero אֶדִיטוֹריאלי — כותרת Serif גדולה, תת-כותרת שקטה, CTA יחיד בולט.

## 10. מיפוי מהקוד הנוכחי → חדש
החלפה גלובלית באובייקט `C` בכל קומפוננטה:
```
purple      #534AB7  →  ink        #37352F   (ראשי)
purpleLight #EEEDFE  →  accentSoft #F3E5DB   (או surfaceAlt #F4F2EC)
purpleMid   #AFA9EC  →  faint      #9B958A
purpleDark  #3C3489  →  inkHover   #211F1A
teal        #0F6E56  →  success    #4F6F52
tealLight   #E1F5EE  →  successBg  #E8EDE6
amber       #854F0B  →  warn       #9A7B3F
red         #A32D2D  →  error      #A6493B
text        #1a1a2e  →  text       #2B2925
muted       #6b7280  →  muted      #6B655C
bg          #f8f7ff  →  bg         #FAF9F6
white       #fff     →  surface    #FFFFFF
border      rgba(0,0,0,0.09) → border rgba(55,53,47,0.10)
```
+ הצגת אקסנט חדש `accent #C2683D` ל-CTA מרכזיים.
+ החלפת gradients סגולים ברקע אחיד `ink` או `accent`.

## 11. סדר ביצוע מומלץ (מחר)
1. טעינת הפונטים (`index.html`) + עדכון `index.css` (font-family בסיס, כותרות Serif).
2. עדכון אובייקט `C` בקומפוננטה אחת מרכזית → לאמת מראה → להפיץ לשאר.
3. החלפת אימוג'ים ב-lucide (קומפוננטה-קומפוננטה).
4. ליטוש: צללים, פינות, מרווחים, כפתורים.
5. landing אחרון — hero אֶדִיטוֹריאלי.

> כשיגיע תיק מותג אמיתי (לוגו/צבעים) — נעדכן את הטוקנים כאן והשאר מתעדכן שיטתית.
